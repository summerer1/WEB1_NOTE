6. 기본 문법-태그 : strong u

HTML에서는 문법적으로 태그라고 부릅니다. 앞에 있는 태그를 열리는 태그, 뒤에 있는 태그를 닫히는 태그라고 구분해서 부르기도 합니다. 닫히는 태그는 태그명 앞에 /를 붙입니다.

strong : 강조표시

u : underline

7. 혁명적인 변화 : h1~h6

h : heading

h1 : 가장 큰 제목 태그

https://www.w3.org/

W3C(The World Wide Web Consortium)라는 국제민간표준화기구의 홈페이지입니다.

h1~h6 : The < h1> to < h6> tags are used to define HTML headings. < h1> defines the most important heading. < h6> defines the least important heading.

9. 줄바꿈 : br p

br : 줄바꿈

br 태그는 닫지 않고 있습니다. img, input, br, hr, meta 등이 닫지 않는 태그의 사례입니다. 쓰는만큼 줄바꿈이 되기 때문에 원하는 만큼 간격을 줄 수 있는 장점

p : paragraph 단락

단락과 단락의 간격이 고정되어 있기 p 태그를 통해서 단락의 경계를 분명히 하면서 CSS를 통해서 p 태그의 디자인을 자유롭게 변경할 수 있기 때문에 br 태그 보다 p 태그가 더 좋은 선택입니다.

10. HTML이 중요한 이유

정보사회에서 HTML을 의미에 맞게 잘 사용하는 것은 비즈니스적인 측면에서 매우 중요합니다. 사용자가 검색엔진에게 coding이라는 정보를 검색했다면 검색엔진은 < h3>로 감싸진 페이지를 더 먼저 보여줄 것입니다. 검색엔진의 검색결과에서 노출되느냐는 실제로 존재하는냐, 존재하지 않는냐의 문제라고 할 수 있습니다.

웹이 중요하게 생각하는 접근성은 신체적인 장애가 있는 사람도 웹을 통해서 정보에 접근할 수 있어야 한다는 것입니다. 웹페이지를 예쁘게 하기 위해서 HTML을 사용하지 않고 웹페이지 전체를 이미지로 만든다면 시각 장애가 있는 분들에게는 암흑과도 같은 상태가 됩니다.
